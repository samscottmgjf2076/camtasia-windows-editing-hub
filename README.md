## Overview

Camtasia is a Windows PC application centered on screen capture and video editing. It keeps recording and post-production in the same desktop environment, helping you go from source footage to a finished video without hopping between separate tools.

This repository presents a desktop-first release that includes LUT handling and export presets. Together, those options give you more influence over how recordings are color-managed and how final exports are prepared for different delivery needs.
